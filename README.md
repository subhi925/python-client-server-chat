# Python Client-Server Chat Application

A simple TCP-based client-server chat application developed in Python.

This project was created as part of my Practical Software Engineering studies and was later improved to strengthen my understanding of networking fundamentals, troubleshooting, and client-server communication.

## Features

* TCP socket communication
* Client-server architecture
* Multi-threaded message handling
* Tkinter graphical user interface
* Basic Caesar Cipher message encryption/decryption
* Real-time message transmission
* Connection handling and message processing

## Technologies

* Python
* Socket Programming
* TCP/IP
* Threading
* Tkinter

## How It Works

1. The client connects to the server using a TCP socket.
2. Messages are encrypted before transmission.
3. The server receives and decrypts incoming messages.
4. Encrypted and decrypted messages are displayed for demonstration purposes.
5. Communication occurs in real time between client and server.

## Learning Objectives

This project helped me gain hands-on experience with:

* Client-server communication
* TCP networking concepts
* Concurrent processing using threads
* GUI development with Tkinter
* Basic encryption techniques
* Debugging and troubleshooting communication issues
